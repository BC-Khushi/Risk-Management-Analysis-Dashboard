# Risk Management Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes organizational risks, risk scores, default probabilities, and key risk indicators to support effective risk assessment, monitoring, and decision-making.

## Tools Used

* Power BI
* Excel

## Features

* Risk Overview Analysis
* Risk Score Analysis
* Loan v/s Risk Monitoring
* Market and Economic Impact Analysis
* Liquidity Risk Tracking
* Default Probability Analysis
* Interactive Filters and Drill-Down Reports

## Key Insights

* Identified high-risk entities using risk scoring models and KPI metrics.
* Analyzed credit, market, operational, and liquidity risks across different categories.
* Monitored default probabilities to support proactive risk mitigation.
* Compared risk levels across regions, industries, and customer segments.
* Visualized risk trends and key indicators to enhance strategic decision-making.
